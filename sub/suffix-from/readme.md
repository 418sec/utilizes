# suffixFrom

 Get suffix from string - by splitter.

**Usage:** `suffixFrom(subject: string, splitter: string | RegExp): string`

```typescript
import { suffixFrom } from 'suffix-from'

suffixFrom(`a/b/cd `, `/`)
// Output: 'cd'
```
